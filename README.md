一键管理所有应用的**运营消息**通知通道，仅支持 MIUI

## 测试环境

> 小米11 Ultra  
> Android 12  
> MIUI 13

## 使用方法

1. 在 Xposed 管理器 (如 LSPosed) 中激活模块
2. 作用域勾选 `系统框架(android)`
3. 重启系统界面或者重启手机

## 无法使用

请先检查模块是否正常激活，并且作用域是否勾选。如果排查后仍有错误，请提交 Issue，并附上 LSPosed 的日志。
